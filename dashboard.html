EDA Recommendations Summary

1. Age Nulls: Impute 174 missing values using median age grouped by Class and Sex.

2. Fare Outliers: Cap extreme fares (max £512.0) at the 99th percentile or apply a log-transform before machine learning models.

3. Fare=0: Double check 15 passengers with £0 tickets (likely crew or promotional comps).

4. Embarked: Impute 2 missing embarkation ports with the mode ('S').
